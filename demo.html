API ======= > Editor.createTable(rows, cols) Creates a new table (replaces existing) with number of rows and columns, filled with random text. > Editor.table Accesses the table API: > new Point(x, y) Creates a new Point > new Rect(new Point(x1, y1), new Point(x2, y2)) Creates a new Rect > Editor.table.select(x, y); > Editor.table.select(x1, y1, x2, y2); > Editor.table.select(point1, point2); > Editor.table.select(point); > Editor.table.select(rect); Selects a single cell (x,y or point), or a range of cells (x1, y1) to (x2, y2), point1 to point2, or rect. To create a point or a rect see the previous entry. > Editor.table.deselect() Removes all selection > Editor.table.mergeCells() Merges the current selection into one cell > Editor.table.splitCells() Splits the current selection into atomic cells (rowspan and colspan all 1) > Editor.table.insertRowAbove() > Editor.table.insertRowBelow() Inserts a row above/below the current selection > Editor.table.insertColumnBefore() > Editor.table.insertcolumnAfter() Inserts a column before/after the current selection > Editor.table.deleteColumns() > Editor.table.deleteRows() Deletes the selected columns or rows, based on the current selection.
